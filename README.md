# LINEBOT

## 環境変数の設定
set LINE_ACCESS_TOKEN=**************  
set LINE_SECRET_TOKEN=**************  

## 以下の手順で動作します。

1. npm install express
1. npm install puppetter
1. npm i @line/bot-sdk express
1. Line Webhook setting
1. ngrok http 3000
1. node server.js

## 下記のキーの設定が必要です。
* channelSecret
* channelAccessToken
